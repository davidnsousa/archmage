# Archmage - Arch Linux Post Installation Setup

Archmage is a simple post-install bash script to choose easily from a variety of packages for daily use from both the official Arch repositeries and AUR using yay. I wrote the script according to my software needs and XFCE preferences (theming, keyboard-shortcuts, etc) to help me replicate or reset my Arch system whenever I need. 

Fork it and ghange it according to your needs.

### Steps

1. Install Arch Linux with __[archinstall](https://wiki.archlinux.org/title/Archinstall)__

2. Clone and run archmage:

```
git clone https://github.com/davidnsousa/archmage
cd archmage
sh archmage.sh
